# TagChat

## プラグイン説明
同じタグを持ったプレイヤーにメッセージを送ることができるようになるプラグイン<br>
指定できるタグはchatから始まるタグのみ(例：chattest、chat1、chat2など)

## コマンド
/tagchat [タグ名(chat:allの場合はそのプレイヤーが指定できるタグ どれかを持っているプレイヤー] [メッセージ]

## 使い方
opの方が、それぞれのプレイヤーにタグをつける <br>
/tagchatコマンドでメッセージ送信
